# Film Tracker (homework one)
A simple c++ program to track the films you've watched.  
  
This program was created for ITCS 2550.

## Features
- Add films with title, release year, rating, and viewing type.
- Print and store a formatted table of all films added.
- Compute average rating.
- Beautiful, easy to navigate menu.

Must be run in visual studio
